## ✨ What Is This?

</div>

> Ever stared at Bubble Sort code and thought *"okay but WHY?"*  
> This project is the answer. Watch **5 sorting algorithms** run live — bar by bar, step by step, swap by swap.  
> No setup. No install. Just open and watch. 🎬

</div>

```yaml
controls:
  - name: Algorithm Picker
    options: [Bubble, Selection, Insertion, Merge, Quick]
    
  - name: Array Size
    range: 10 → 80 elements
    
  - name: Speed Control
    range: Very Slow → Very Fast
    
  - name: New Array
    action: Generate fresh random array anytime

stats:
  - Comparisons (live counter)
  - Swaps (live counter)
  - Time Complexity display
  - Algorithm description card
```

## 📊 Complexity Cheatsheet

</div>

| Algorithm | Best 🌟 | Average 📊 | Worst 💀 | Space 🧠 |
|:---------:|:-------:|:----------:|:--------:|:--------:|
| Bubble Sort | `O(n)` | `O(n²)` | `O(n²)` | `O(1)` |
| Selection Sort | `O(n²)` | `O(n²)` | `O(n²)` | `O(1)` |
| Insertion Sort | `O(n)` | `O(n²)` | `O(n²)` | `O(1)` |
| Merge Sort | `O(n log n)` | `O(n log n)` | `O(n log n)` | `O(n)` |
| Quick Sort | `O(n log n)` | `O(n log n)` | `O(n²)` | `O(log n)` |
